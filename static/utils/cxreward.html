Cathay Pacific Reward Availability

This tool gives approx award availability for CX flights ONLY (i.e. no codeshare/partner flights). It uses the unauthenticated query API (e.g. when you browse reward flights on Cathay website without logging in).

The main benefit is to quickly explore availability across different routes, classes & dates when planning a flexible getaway.





Protip: Hover over a certain cell to see the cost breakdown and value per mile
